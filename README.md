# Customer_churn_prediction
Customer Churn Prediction
Overview: This project predicts customer churn using machine learning models and identifies key factors driving customer attrition.

Business Problem
Customer churn is a major challenge for subscription based businesses. The goal of this project is to identify customers at risk of leaving and provide actionable insights to improve retention strategies.

Models Used
Logistic Regression (with class balancing)
Random Forest Classifier

Evaluation Metrics
Accuracy
Precision
Recall
F1 Score
ROC AUC

Key Insights
Customers with higher monthly charges are more likely to churn
Longer tenure reduces churn risk
Fiber optic users show higher churn rates

Business Recommendations
Focus on early customer retention (first few months)
Offer incentives for high monthly charge customers
Improve experience for fiber optic users
Use model predictions for proactive retention campaigns

Tools & Technologies
Python, Pandas, NumPy, Scikit-learn, Matplotlib / Seaborn

Project File
Churn_Project.ipynb
